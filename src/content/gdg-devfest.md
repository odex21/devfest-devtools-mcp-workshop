我参加的是 **Google DevFest 2025 深圳开发者嘉年华（AI / 出海 / Android / Web）**。  
主论坛的主题是「未来已来，探索 Google 科技最前沿！」，分享了 AI 赋能未来、Gemini & Gemma 模型、Web 前沿技术（Web3 / PWA / WebAssembly）、Android 生态创新、Google Cloud & Firebase 等方向。  
在 **GDG** 社区，你会发现很多熟悉的关键词：**Android、Web、Cloud、AI**，也会发现很多不那么陌生的人——  
大家来自不同的公司、学校和城市，但在 DevFest 的现场，大家有一个共同的身份：**「热爱分享与学习的开发者」**。

我参加的这个 Workshop 属于 **Web 分会场**，以 **Chrome DevTools MCP Server** 为主题，从浏览器调试到自动化、从本地开发体验到 MCP 协议和工具生态，  
让我对「**浏览器 + 工具链**」之间的连接有了更加直观的认识。  
这份页面不是官方活动页，而是我自己的 DevFest Workshop 记录。

### GDG 是什么？

- 由本地志愿者组织的 Google 开发者社区  
- 专注技术分享、线下活动和实践

### DevFest 的特点

- 一年一度的开发者节日  
- Talk + Workshop + 交流 & Networking

### 我在这场 DevFest Workshop 里的节奏

1. **认识 GDG 与 DevFest**：社区介绍、往期活动回顾  
2. **Chrome DevTools MCP 概览**：为什么需要 MCP？能做什么？  
3. **实战 Workshop**：配置 DevTools MCP Server，完成端到端联动  
4. **自由交流**：和讲师、同学交流技术实践与踩坑经验
